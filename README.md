Files adapter for [Ruby Object Mapper](https://github.com/rom-rb/rom).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rom-files'
```

And then execute:

```bash
bundle
```

Or install it yourself as:

```bash
gem install rom-files
```
