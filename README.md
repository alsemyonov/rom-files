Filesystem adapater for [Ruby Object Mapper](https://github.com/rom-rb/rom).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rom-filesystem'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rom-filesystem
